## 💡 The Problem
Traditional security tools are static and miss complex business logic flaws. Penetration testing is slow and manual. Aura solves this by automating the most difficult parts of offensive security with neural state modeling.

## 🚀 Key Features
- **Neural State Modeling:** Learns terminal-like target logic to navigate complex stateful workflows.
- **Autonomous Weaponization:** Automatically generates functional Python PoC scripts for detected critical flaws.
- **Stateful Logic Fuzzer:** Executes path-aware fuzzing across hundreds of dependency-mapped nodes.

## 🛠️ Built With

### Languages
- **Python (3.10+):** The primary engine for logic, strategic decision tracking, and autonomous weaponization.
- **Go (Golang):** Powering the **Nexus High-Speed Networking Core** for ultra-fast parallel port scanning and raw socket manipulation.
- **HTML5/CSS3/JavaScript:** Used for the mission dashboard and reporting engine.

### Frameworks & Libraries
- **Rich:** For the immersive, professional-grade terminal UI and logging.
- **Httpx & Aiohttp:** Asynchronous HTTP/2 and networking client for high-concurrency probing.
- **Esprima-Python:** Used in the **Semantic AST Engine** for deep JavaScript taint analysis.
- **Transformers/HuggingFace:** Powering some of the neural state models for strategic decision making.

### Techniques & Protocols
- **Directed Acyclic Graphs (DAG):** For stateful workflow dependency mapping.
- **JSON-RPC 2.0:** For the bridge between the Python brain and the Go-networking core.
- **LLM-Driven Strategic Mutation:** Using AI to predict the next best attack vector based on target feedback.
